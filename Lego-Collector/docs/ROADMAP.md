# 🗺️ Roadmap: LEGO Collector

## Vision
Att skapa en sömlös integration mellan Brickset och Obsidian för att hantera och visualisera en personlig LEGO-samling med metadata som är kompatibel med både Obsidian och framtida Notion-export.

## Faser

### Fas 1: Grundläggande Datahämtning (Slutförd ✅)
- API-koppling.
- Automatisk generering av Markdown-filer.
- Lokal bildhantering.

### Fas 2: Visualisering & Organisation (Slutförd ✅)
- HTML-Dashboard för överblick.
- Metadata-schema för Obsidian-kompatibilitet.

### Fas 3: Samlingshantering & Tvåvägssynk (Slutförd ✅)
- [x] Lägg till set i Brickset direkt från CLI (`--own`, `--want`).
- [x] Smart bulk-import med automatisk arkivering i listan.
- [x] Dubblettkontroll mot lokalt valv.

### Fas 4: Ekosystem-integration (Planerad 📅)
- [ ] Notion-synkronisering (direkt API eller export-guide).
- [ ] Synkning av lokala anteckningar (Skick/Byggat) till Bricksets "Notes"-fält.
- [ ] Automatiserade backups av datan.
